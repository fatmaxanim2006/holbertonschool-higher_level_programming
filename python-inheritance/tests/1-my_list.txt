The ``MyList`` module
======================

Using ``MyList``
----------------

Importing the class:
    >>> MyList = __import__('1-my_list').MyList

Checking normal list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Checking with negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-5)
    >>> my_list.append(0)
    >>> my_list.print_sorted()
    [-5, -1, 0]

Checking with empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
